apsis
===
[![npm version](https://badge.fury.io/js/apsis.svg)](https://badge.fury.io/js/apsis)

npm package for apsis newsletter marketing service

## Getting Started

Install using npm:
```
npm install apsis
```

Require and initiate in your project:
```js
const apsis = require('apsis');

apsis.init({
  ApiKey: xxx
});
```
